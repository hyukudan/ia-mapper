# Codebase Navigation

## Entry Points

- `skills/mapper/SKILL.md`: primary orchestration instructions for Claude Code.
- `skills/mapper/scripts/scan-codebase.py`: scanning + caching + module hashes.
- `skills/mapper/scripts/plan-assignments.py`: grouping by token budgets.
- `skills/mapper/scripts/risk-signals.py`: risk report generation.

## Common Tasks

**Update defaults**
- Edit `.mapper.json.example`, then mirror changes in `.mapper.json`.

**Add a new helper script**
- Place it in `skills/mapper/scripts/`.
- Document it in `skills/mapper/SKILL.md` and `README.md`.

**Run a lightweight pass**
- Generate a skeleton with `skills/mapper/scripts/skeletonize.py`.

**Publish the plugin**
- Ensure `.claude-plugin/plugin.json` is accurate.
- Push to GitHub and register in the Claude Code plugin registry.

## Outputs

- `docs/CODEBASE_MAP.md`: full map and architecture summary.
- `docs/CODEBASE_NAV.md`: this quick guide.
- `docs/RISK_SIGNALS.md`: risk summary (generated by `risk-signals.py`).
